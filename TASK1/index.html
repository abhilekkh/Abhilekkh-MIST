HTML Basics

What is HTML?

  1. HTML stands for HyperText Markup Language.
  2. It is the standard language for creating webpages.
  3. HTML uses tags to structure content.
  4. It is not a programming language, but a markup language.

Basic Structure of an HTML Document: 


 <<!DOCTYPE html>
 <html lang="en">
 <head>
     <meta charset="UTF-8">
     <meta name="viewport" content="width=device-width, initial-scale=1.0">
     <title>Page Title</title>
 </head>
 <body>
     <h1>Hello, World!</h1>
     <p>This is my first HTML page.</p>
 </body>
 </html>

Explanation: 

<!DOCTYPE html> ‚Üí Declares HTML5 version.
<html> ‚Üí Root element of the page.
<head> ‚Üí Contains metadata (title, links, scripts, styles).
<body> ‚Üí Contains visible content (text, images, links, etc.).

HTML Elements and Tags

  1. Element:Everything from opening tag to closing tag.
  2. Tag:The markup used to define elements.

Example: 


 <p>This is a paragragh.</p>
  • <p> ‚Üí opening tag
  • </p> ‚Üí closing tag
  • This is a paragragh ‚Üí content

Self-closing tags: Tags that don't have closing tags. 

<img src="image.jpg" alt="Sample Image"> 
<hr>
<br>

CSS Basics

CSS is used to style web pages. Example:

h1 {
    color: blue;
    text-align: center;
}

Some common properties:

  • color ‚Üí text color
  • background-color ‚Üí background color
  • font-size ‚Üí text size
  • margin / padding ‚Üí spacing
  • border ‚Üí border style
