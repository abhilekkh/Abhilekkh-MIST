HTML Basics

What is HTML?

  1. HTML stands for HyperText Markup Language.
  2. It is the standard language for creating webpages.
  3. It is not a programming language, but a markup language.

Basic Structure of an HTML Document: 


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Page Title</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

Explanation: 

<!DOCTYPE html> → Declares HTML5 version.
<html> → Root element of the page.
<head> → Contains metadata (title, links, scripts, styles).
<body> → Contains visible content (text, images, links, etc.).

HTML Elements and Tags

  1. Element: Everything from opening tag to closing tag.
  2. Tag: The markup used to define elements.

Example: 


<p>This is a paragraph.</p>
  • <p> → opening tag
  • </p> → closing tag
  • This is a paragraph → content

Self-closing tags: Tags that don't have closing tags. 

<img src="image.jpg" alt="Sample Image"> 
<hr>
<br>

HTML Headings

  1. Headings define the importance of text.
  2. HTML supports 6 levels: <h1> to <h6>

<h1> Heading 1 </h1> <!--Largest-->
<h2> Heading 2 </h2>
<h3> Heading 3 </h3>
<h4> Heading 4 </h4>
<h5> Heading 5 </h5>
<h6> Heading 6 </h6> <!--Smallest-->

Paragraphs and Text Formatting

  • Paragraph: 
    <p>
  • Line Break: 
    <br>
  • Horizontal Line: 
    <hr>

Text Styles:

  • Bold: 
    <b>
    or 
    <strong>
  • Italics: 
    <i>
    or 
    <em>
  • Underline: 
    <u>
  • Subscript: 
    <sub>
  • Superscript: 
    <sup>
Example: 
<p> This is <b>bold</b> and this is <i>italics</i>.</p>
<p>H<sub>2</sub>O is water </p>

Links (Hyperlinks)

  • Tag: 
    <a> (anchor)
  • Attribute: 
    href
    → URL of link

Other attributes: 

  • target="_blank"
    → opens in new tab
  • title="..."
    → shows tooltip on hover

Images

  • Tag: 
    <img>
    (self-closing)
  • Attributes:
    • src
      → image URL
    • alt
      → alternate text (for accessibility)
    • width
      and 
      height
      → size

<img src="image.jpg" alt="My Image" width="200" height="150">

Lists

  1. Ordered List 
    <ol>
    → Numbers items automatically.

    2. <ol>
  <li> Item 1</li>    
  <li> Item 2</li>   
</ol>

  2. Unordered List 
    <ul>
    → Bullets items.

    3. <ul>
  <li> Apple</li>    
  <li> Banana</li>   
</ul>

  3. Nested Lists

    4. <ul>
  <li> Fruits</li>    
    <ul>
      <li> Apple</li>    
      <li> Banana</li>   
    </ul>
  </li>
</ul>

Tables

  • Tag: 
    <table>
  • Table row: 
    <tr>
  • Table header: 
    <th>
  • Table data: 
    <td>

Example: 

<table border="1">
  <tr> 
    <th>Name </th>
    <th>Age </th>                                                            
  </tr>   
  <tr> 
    <td>John </td>
    <td>25 </td>                                                            
  </tr>
  <tr> 
    <td>Mary </td>
    <td>30 </td>                                                            
  </tr> 
</table>

Forms

  • Tag: 
    <form>
  • Common input types: text, password, radio, checkbox, submit

<form action="/submit" method="post">
 Name: <input type="text" name="name"><br> 
 Gender:    
  <input type="radio" name="gender" value="male"> Male                                                            
  <input type="radio" name="gender" value="female"> Female <br>  
  <input type="submit" value="Submit"> 
 </form>
Form attributes: 

  • action
    → URL where data is sent
  • method
    → 
    GET
    or 
    POST


HTML Semantic Tags

  • <header>
    → top of page section
  • <footer>
    → bottom of page section
  • <nav>
    → navigation links
  • <main>
    → main content
  • <section>
    → section of content
  • <article>
    → independent, self-contained content
  • <aside>
    → sidebar content

Multimedia

  1. Audio

    2. <audio controls>
  <source src="audio.mp3" type="audio/mpeg">
</audio>

  2. Video

    3. <video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
</video>

CSS Basics

CSS is used to style web pages. Example:

h1 {
    color: blue;
    text-align: center;
}

Some common properties:

  • color → text color
  • background-color → background color
  • font-size → text size
  • margin → spacing
  • padding → spacing
  • border → border style
